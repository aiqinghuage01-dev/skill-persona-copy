# 人设文案模板库

> 第一人称视角 — 老板自己讲自己的故事
> 按行业分三大类：工厂类、店铺类、服务咨询类

---

## 一、工厂类模板（6个）

### 模板1："外表风光，背后凄凉"

**适用行业**：制造业、生产型工厂
**核心钩子**：看似年产值几千万/几个亿，实际存款不如员工
**结构**：反差数字开场 → 逐项拆解成本（原材料/库存/设备） → "外表风光，背后凄凉"收口

**范例原文**：

每个老板啊，都说他自己年产值有几千万几个亿，那他兜里到底有多少存款呢？就大我来说了，我是我们工厂的生产副总，我们工厂是专门生产大件塑料塑纳用品的，去年的年产值是2.6个亿，看着表面很风光的样子，但是你们知道我有多少存款吗？我怕你们的笑话，我身上全部加上掏出来就1万多，真的还不如他们的销售工存款多啊。可能有些朋友会问，你的钱到底哪里去了？首先啊，我们做塑料制品的朋友们应该都知道，我们正好就是薄利多销，利润少的可怜。其次呢，我们工厂有近80台的这种机，还有各种火机啊，模具啊，这些都是看不见的钱啊，这看看我们仓库的成本，库存有3000多万，还有我们啊原材料有将近5000万的库存，这些都是开工厂必不可少的东西，所以啊，朋友们，我们看人不要只看表面，有些人他看着好像是很有钱的，但满人就是个纸老虎，外表风光，背后凄凉。

---

### 模板2："白手起家，事业初心"

**适用行业**：门窗、建材、加工制造
**核心钩子**：从高中毕业放羊到万平工厂
**结构**：年份线索串起创业历程 → 强调品质初心 → "先入眼再走心"收口

**范例原文**：

我叫高雪雄，80后，白手起家的创业者，2000年高中毕业步入社会，在老家放过羊，养过羊，进过厂，打过杂，也迷茫过。一二年的时候，我接触到门窗这个行业，仅仅4个月时间就让我有了开门窗厂的想法，于是在2013年开了一家门窗小作坊。从一三年开始到现在，从200平方到现在2万多平，我们也经历了好几次厂房的更新，这一路走来也很不容易。创业这些年，我们一直在思考公司的发展，一刻都不敢停，经常去外地与同行交流学习，提高自己的能力。我一直相信天道酬勤，事在人为，因为我们坚持质量，以品质征服人心。产品和人一样，先入眼再走心，一见钟情是颜值，最后经验的才是品质。

---

### 模板3："摸爬滚打，诚信服务"

**适用行业**：定制加工、工程项目类
**核心钩子**：从行业小白到专业人士的蜕变
**结构**：入行经历 → 各岗位历练 → 服务初心金句 → "工程无大小，质量无小事"

**范例原文**：

大家好，我是老邓，是一个90后岛台工厂的创业者，目前经营着一家岛台工厂，很高兴能在抖音里认识大家。13年开始接触野蛮倒台，从最简单的岗位做起，跟单销售、图纸设计、网络营销、工程管理，从一个懵懂的行业小白到现在大家眼里的专业人士，其实在你看不到的地方，面临更多更新更多的挑战，在这个日新月异的大时代，对每个实体经营者来说都是一个严峻的考验。认为开场当老板都很聪明，很有钱，很有面子。其实一路走来，只有自己最清楚，都是踩着坑过来的。真诚坚持，愿意付出是我的人生线条。工程无大小，质量无小事，耐心做服务是我创业的初心。无论你是100方的小工程，还是10万方的大项目，都需要严格把关质量。希望你也和我一样，不随波逐流，但仍然选择相信，保持自己那份清澈。我是老邓，专业定制岩板岛台，如果你有需要，请记得找我。

### 模板4："奋斗成金，品质为王"

**适用行业**：家具、家居定制
**核心钩子**：穷小子600块起步到25000平工厂
**结构**：起步反差 → 拼命跑客户 → 全流程品质把控 → 售后回访 → 苦尽甘来

**范例原文**：

一个穷小子翻身做家具的十年，从刚开始的600块到现在的6000平方的家具展厅和25000平方的生产车间，18岁那年，为了证明自己，背井离乡来到广东龙乡做起了家具行业，我们总活在憧憬理想的人生中，可现实里并不是什么都一帆风顺，常常到深夜，脑子里只有一件事情，就是要赚钱，开始玩命的跑客户、销售、学习、出差，所有的事情亲力亲为，因为我深知只有提升自己，才能得到更多客户的信任，才会选择我们的家具，哪怕客户只买了一个沙发，我也会从焊接、打磨、抛光、框架、图件、鉴别、裁皮、车材、嫩皮，从原材料到产品落地，都会把更好产品的品。同时我也经常去选择我们家具的客户家违坊，看看客户家具用的怎么样，有没有不好的地方需要改进，只有吸取更多的经验，才能做出好的家具产品。很多人说我傻，每天工作从不一切好在不急功近利，付出总会有回报。家具产品订单开始爆满，算是苦尽甘来。为了满足到客户更多的个性化需求，每年都要开发上百个新款产品，到现在展现了一应俱全，意识到卖好品质，卖好服务又平价的家具才能做大。从一个农村的打工仔到现在25000方的家具工厂，我用了10年。我叫小明总，如果最近想选家具，不妨到我们展厅参观，我在展厅期待你的到来。

### 模板5："纸上谈兵，利润微薄"

**适用行业**：印刷、包装、低利润制造业
**核心钩子**：年产值1000万，一年利润就赚个废纸钱
**结构**：抛问题 → 逐项列成本（厂租/水电/工资/辅料/运费/伙食/社保） → 扣除后所剩无几

**范例原文**：

一个年产值1000万的印刷厂，一年的利润到底有多少钱呢？根据当下的行情，一年下来也就赚个费纸钱。也许大家觉得这是开玩笑，那么接下来我们就来算一笔账，一个年产值1000万的印刷厂需要的厂房面积大概是1500到2000个平方，那么厂租一年大概就是40万，水电费也大概要接近40万，那么就去了80万，1000万产值的印刷厂，大概需要配到20个人，按照我们印刷行业的平均工资标准来看的话，那么每个人大概就是7000块钱，一年下来的工资大概就是在170万左右。再就是辅料，就是我们生产过程中用的油墨、油、胶膜、胶水、刀模、嗯、蜂香胶啊、围模等各种辅料，一年大概需要用到70万。再就是咱们的运费，每个月2万块钱，那么一年就是24万。接着就是伙食费加社保，一年大概也就是24万，再就是咱们的一个业务费，一年就算十个。那么我们按照毛利率40%来算，在印刷行业就已经算是比较偏高的了，那么一年的毛利率就是400万，400万减掉我们刚刚算下来的这一些开销，大概就是380多万，那我们还余下十几万，这个十几万里边还有一些没有算到，就是一个机器的折旧费，一些损耗，还有一些因为不小心造成的退货返工，还有机器维修换配件等这一些无法计算的一些费用。所以说一个印刷厂一年下来做1000万，你能赚到一个废纸钱就已经算是不错了。

### 模板6："颠覆传统，品质升级"

**适用行业**：包装印刷、产品升级型制造
**核心钩子**：50万个纸盒赚不到2000块，但意外发现新商机
**结构**：大订单但没利润 → 意外沟通发现客户痛点 → 推荐升级工艺 → 决定转型升级

**范例原文**：

50万个纸盒，你知道我赚了多少吗？我自己的案例啊，就这个案例让我彻底明白了，颠覆了我以前的想法，也改变了我公司的战略，不再纠结价格战，脱离行业内卷。这个客户啊，也是网上找到我的，说他们单款下单的数量是100个W，首次合作呢，少做一点数量就50个W，真的是既高兴又失望，高兴的是来了一个长期的大客户，失望的是我连2000块都没赚到，但是不经意的一个聊天让我赚到了好几个2000。他们在北方定制的包装盒啊，都是普通的卡纸盒，工艺也大多是压纹、烫金、UV等常见的工艺，它一直要脱离同质化的包装，并且要求生产工期要。因为直播一般都不压库存，我就推荐了银卡纸逆向UV印刷，各种逆向纹路都可以设计，这个工艺啊，替代了传统的UV烫金，可以说是印刷UV烫金效果一播出省时还省钱，再高级一些呢，可以加一些3D浮雕热压纹，妥妥的高级感新品啊，我帮他设计打样，客户特别满意，整体货期也比他原来快了2天，从那天起啊，我就决定不再选普通卡纸盒了，就做产品升级，研发更加有品质的工艺。全国市场那么大，需要升级的产品太多了，追求高速度、高品质、高性价比也是必然发展的趋势。如果你需要样品，评论区留言我寄给你。

## 二、店铺类模板（7个）

### 模板7："疫情梦碎，奋斗不止"

**适用行业**：餐饮、实体门店
**核心钩子**：火锅店老板清晨五点逛批发市场
**结构**：反差日常（别人逛商场，我逛菜市场） → 疫情打击 → 转型直播 → 绝不后悔

**范例原文**：

大家好，我是七七一个火锅店的老板，你见过清晨五点的批发市场是什么样的吗？我见过，我曾经是那里的常客，只为了采购到最新鲜的食材。当别的漂亮女孩在逛商场的时候，我通常在逛农贸市场，只为了询价拿到最好的食材和最低的价格。当别人家的男朋友刷卡买包的时候，我在门店因为几块钱的小事儿跟客人赔礼道歉。疫情再度深持续留在影响。2020年疫情大爆发，对所有的实体有了致命的打击，我们也不例外。我曾整夜睡不着觉，房租付不上，工资也发不出。没有办法，我做起了直播，开始了线上卖货。有人问我做餐饮累吗？我说累呀，那这么累，你后悔吗？我后悔，我后悔，我还不够努力。人生的价值不在于享受了多少，而是在于经历了多少。外在的皮囊总会逝去，内在的灵魂才会不断升华。现在我的火锅店一天2万的营业，我还在不断努力，只为了实现自己最大的价值，你呢？在行动了吗？

---

### 模板8："学徒翻身，梦想实现"

**适用行业**：餐饮连锁、技术型门店
**核心钩子**：从学徒到四家火锅店老板，用了17年
**结构**：起点（学徒/打杂） → 换7份工作被质疑 → 求学+开第一家店 → 疫情考验 → 金句收口

**范例原文**：

从一个学徒到现在成为四家火锅店的老板，我用了17年的时间。我叫吴玉星，今年38岁，我是科班出身。毕业以后我就孤身一人去了成都，从饭店学徒干起，一年的时间我换了7份工作，他们都说我太年轻，干工作没有长性，成不了事儿。我心里想的是燕雀安知鸿图之志。那一年，脏活累活全干了，世间百态也看了，人情冷暖也长了，所以落魄的时候连个睡觉的窝都没有。忽然有一天，我发现这些都不是我发展的方向，我毅然出走。我四处求学，我要开一家属于自己的火锅店，一个和任何人都不一样的店。直到2010年，我开了属于自己的第一家火锅店。为了节省开支，我就住在店里，一张破门板，几块砖头，垫起来就是帐虫。那个时候虽然苦，但是心里很甜，因为我看到梦想正在一点点实现。直到后来的几年，我又陆续开了三家店。其实真正困难的时候就是过去的三年，房租、人工，所有成本开支压得我喘不过气来。但命运不会辜负每一个努力奔跑的人，我利用那个时间努力丰富自己。我曾经也把失败归结为时运不济，但这次创业让我明白了，这个社会比的从来不是谁比谁运气好，而比的是谁比谁更有韧性。荀子说过，道阻且长，行则将至，行而不辍，未来可期。当你跌入谷底，万念俱灰时，希望这个视频可以给你力量。

### 模板9："创业故事，文化传承"

**适用行业**：餐饮品牌、文化传承类
**核心钩子**：没读过书的创业者，从卖东西到建品牌
**结构**：苦出身 → 第一桶金 → 品牌诞生 → 女性创业不易 → 文化使命

**范例原文**：

大家好，我是朱博芬姐。回想起从前，那时候条件不好，没有读过书，没有出路，在市场卖过东西，端过盘子，那时候的我只想尽力的把能做的事情做好，我相信只要肯学，总要有出路，也许老天爷也眷顾不服输的想改变命运的人。我用将来的第一桶金，再加上开了属于自己的第一个饭店，由于往来的经验之上，我独立研发了一篇菜品，也是从才起。猪阿婆三个字代表了客户对我的信任，代表了我，也成就了我。也是从这个时候开始，XX 正式出现大家的视野里。那时也并非一帆风顺，最难的时候我生完孩子还没坐月子的时候就投入了工作，作为女性创业者，有很多无奈的都是说不出来的。后面我们受到了越来越多的认可，朱肉婆三个字也让更多的人记住了。天悦集团顺势而起，很多人都说猪肉婆够了，不要再做，但是在我心目中没有终点的挑战。一句话是这样说的，组织社会，回归社会，除了做好家乡的菜，还要坚持弘扬我们传统的文化。希望通过这个平台，让我们顺德美食跟传统文化出广东，走出全世界。在接下来的日子里，我们一起努力，想开什么内容啊，关注我，我在评论区等你呀。

### 模板10："诚信经营，顾客至上"

**适用行业**：美业、服务型门店
**核心钩子**：10年老店，靠口碑说话
**结构**：短小精悍 → 直接表态不玩套路 → "顾客是最好的裁判"

**范例原文**：

我是红姐，在北京已经开了10年的美容院了，我一直认为做美业还是要良心一点，不要去动什么坏心思，更不要玩什么套路，顾客又不是傻子，你的服务好不好，品质高不高，价格合不合理，顾客心里都是有数的。所以我一直认为顾客就是最好的裁判，口碑就是最好的检验。这也是我这10年来在北京真诚的去服务好每一位顾客的原因。北京的姐妹们欢迎来红姐的小店做客

### 模板11："实力为王，老板娘坚持"

**适用行业**：餐饮、女性创业者
**核心钩子**：别人说生意好是因为老板娘好看
**结构**：回应质疑 → 拆解餐饮真正的功夫（采购/研发/运营） → "留住顾客的是口味不是颜值"

**范例原文**：

多网友嘲讽我，说我店里生意好，完全是因为老板娘好看。每当我听到这种说法，真的是又高兴又无奈，高兴的是他们真的有在夸我漂亮，无奈的是，一个餐饮门店要想在现在这种竞争激烈的环境中脱颖而出，从食材采购、产品研发到推广宣传、门店运营，每个环节都必不可少，每个环节也都至关重要。餐饮店能留住顾客的从来不是谁的颜值，一定是菜品的口味儿。虽然他刚刚接触餐饮，但我会一直把产品本身放在首位，不会本末倒置，更不会做一锤子买卖。

### 模板12："味道认证，全程教学"

**适用行业**：餐饮培训、技术传授型
**核心钩子**：从88年做火锅，7家直营店
**结构**：资历背书 → 先尝味道再决定 → 全流程教学承诺 → 毫无保留

**范例原文**：

大家好，我是幺舅，如果你想开一家重庆老火锅店，但是没有技术，可以来找我，我教你。我从88年开始做火锅，在重庆有7家直营店，如果你想学，可以先到我的任何一家门店去品尝味道，认可我的味道后再来找我，就在我的实体店里面学，从早上开门到晚上关门，从买原材料到炒料到配菜到店铺经营，全部都可以学，所有的配方配料都是毫无保留的传授，而且这里不限时间，学会为止。如果有兴趣可以私信我。

### 模板13："辛苦有回报，客满如潮"

**适用行业**：餐饮、小本创业
**核心钩子**：15万开店，日营业额冲到29000
**结构**：投入金额 → 忙碌日常 → 虽苦但值 → "一天赚到之前半年工资"

**范例原文**：

大家好，我叫赵刚，35岁的我用这些年存下的15万块钱鼓起勇气开了这家火锅店。昨天很累，但是很开心，因为昨天的营业额竟然冲到了29000。虽然是街边小店，他每天都是客满，低谷时段营业额也能够稳定在一万七八。我们差不多早上10点半开始营业，客人紧接着就来了，我的火锅生活也开始了。我每天都是打仗的状态，不停的接待客人，清理桌面，上菜，常常一站就是好几个小时。确实每天都很辛苦，但是这种忙碌啊，比起朝九晚五的工作充实多了。我们的火锅不仅是视觉上吸引人，让人一看就想吃，干这行辛苦是难免的，当你看到你一天的营业人就能赚到你之前半年工资的时候，这一切都值了。
